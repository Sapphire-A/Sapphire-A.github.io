TOOLS AND ANALYSIS

This project will use the following statitical tools to analyze the data:
Regression - Time-Series - Least-Squares - Forecasting
Correlation - ANOVA - Chi-Square - F-test

CORRELATION

Correlation is one of the tools used in regression. The coefficient of correlation R, States the strength of the relationship whereas R2, the coefficient of determination explains by how much this change is due to the independent variable, which is the percentage of values that actually fall on the trend line.

REGRESSION

Regression analysis is a set of statistical methods used to determine and estimate relationships between variables of interest, but specifically between a dependent variable and one or more independent or explanatory variables. This is done by forming a line of best fit to the data values otherwise known as a trend line.

A regression model shows whether changes observed in the dependent variable are associated with changes observed in one or more of the independent variables.

One of the main purposes of regression is to predict unknown and future values of the dependent variable, thus, it is used in forecasting. Regression uses Chi-squared test of independence as a tool which in turn uses the F-test to ascertain probabilities for its hypothesis calculations.

There are various methods of trend line fitting including the Least Squares Method. This is the preferred technique that will be used for this project.

TIME SERIES

A time series is set of observations ordered with respect to the successive time intervals. In other words, it is the chronological arrangement of data. Time is an independent variable and can be in hours, days, months or years.

A time series depicts the relationship between two variables of which Time is the independent and the other , is any quantitative dependent variable. Th components of Time Series are the Trend, the Seasonal Variations, the Cyclical Variations and Random movements and its models include the Additive, the multiplicative and the mixed models. The most important use of time series is to predict future values of the dependent variable.

This project shall be using the Time Series of the actual Happiness index Scores of 128 selected countries from 2013 to 2019 for its calculations, and shall use the Additive model Y = BX + C as it is a linear relationship between the scores and time

THE LEAST SQUARES METHOD

Least Squares is a method for finding the best fit of a set of data points by minimizing the sum of the residuals of points from the plotted line or curve. It gives the trend line of best fit to a time series data in such a manner that the following two conditions are satisfied.
  • The sum of the deviations of the actual values of Y and the computed values of Y is zero.
  • The sum of the squares of the deviations of the actual values and the computed values is least.
The trend line (Y) is defined by the following equation:

Y = BX + C

Where, Y = predicted value of the dependent variable

C= Y-axis intercept; when X = 0, Y = C

B = slope of the line (the rate of change in Y for a given change in X)

When B is positive the slope is upwards, and when it is negative, the slope is downwards

X = independent variable (in this case it is time)

To estimate the constants B and C, the following two equations have to be solved simultaneously:

∑Y = nC + B∑X

∑XY = C∑X + B∑X²

To simplify the calculations, the midpoint of the time series is set to be zero (origin) so that the negative values in the first half balance out the positive values in the second and ∑X = 0.

FORECASTING

Forecasting is a technique of predicting future events, values or observations based on the results of previous data statistically perused. There are two basic types: Quantitative and Qualitative forecasting Within which many technique abound including the Time Series method, the Econometric and the Straight-line method among others.

This project shall build upon the Time Series model using the Least Squares technique already applied. The Years in view for forecasting are 2023 to 2028, a time series span of six years.

CHI-SQUARED

Chi-square (χ2) statistic is a test that measures how a model compares to actual observed data. The purpose of this test is to determine whether there is a statistically significant difference between them both and if so, to determine if it is due to chance, or if it is due to a relationship between the variables under study.

It is used to test hypothesis using set degrees of freedom and set levels of significance There are two types of chi-square: The goodness of fit test which determines if sample data fits a population; and the test for independence which determines if two distributions of categorical variables differ from each other. F-TEST

An F-test is any statistical test in which the test statistic has an F-distribution under the null hypothesis. It is mostly used for the comparison of statistical models that have been fitted to data sets, to identify models that best fit population data. Its other uses include testing equality of variances of two population, and testing equality of several means.

It will be used to test the significance of the regression model. In other words to test whether or not the linear regression model provides a better fit to the dataset than a model with no predictor variables. The F-Statistic is a ratio of two variances.

The F Value is calculated using the formula F = (SSE1 – SSE2 / m) / SSE2 / n-k, where SSE = residual sum of squares, m = number of restrictions and k = number of independent variables. the F Statistic which is the critical value for the test) can be found in the F-Table.
